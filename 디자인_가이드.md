## 4. GNB (Global Navigation Bar)

### 4-1. 구조: 2단 레이아웃

```html
<nav id="app-nav">   <!-- 또는 <header id="main-header"> -->

  <!-- 1행: 좌(허브 이동) + 우(다크모드 토글) -->
  <div class="nav-top-row">
    <a class="gate-btn" href="../index.html">
      <i class="ri-home-4-line"></i>
    </a>
    <button class="dark-btn" onclick="toggleDark()">
      <i class="ri-moon-line" id="dark-icon"></i>
    </button>
  </div>

  <!-- 2행: 앱 타이틀 + 서브타이틀 -->
  <div class="nav-title-section">
    <div class="nav-main-title"><i class="ri-..."></i> 앱 이름</div>
    <div class="nav-sub-title">안내 문구</div>
  </div>

  <!-- 3행(선택): Pill Tab -->
  <div class="nav-tabs">
    <button class="nav-btn active" onclick="showPage('tab1')">탭1</button>
    <button class="nav-btn" onclick="showPage('tab2')">탭2</button>

</nav>
```

### 6-3. JS 토글 패턴

```js
function showPage(name) {
  // 모든 .page 비활성화
  document.querySelectorAll('.page').forEach(p => p.classList.remove('active'));
  document.getElementById('page-' + name).classList.add('active');

  const isDetail = ['study', 'test', 'book', 'bigtest'].includes(name);
  const nav = document.getElementById('app-nav');      // GNB
  const dh  = document.getElementById('detail-header');

  if (isDetail) {
    nav.style.display = 'none';
    dh.classList.add('visible');
    document.getElementById('detail-title').textContent = pageTitles[name];
  } else {
    nav.style.display = '';
    dh.classList.remove('visible');
  }
}
```

## 7. 앱별 상세 구조

### 7-1. 영어학습_앱.html

| 구분 | 내용 |
|------|------|
| GNB | `<header id="main-header">` |
| 탭 | 단어 / 본문 / 시험 (3탭) |
| 상세 | 단어학습·본문학습·시험 진입 시 main-header 숨김, detail-header 표시 |
| detail-header 제목 예시 | "Unit 1 단어" / "Unit 1 본문" |
| 내부 탭 (본문) | 스크립트 / 문법 + 한국어 숨기기 토글 |

**화면 전환 흐름**
```
홈(단원 그리드) → 단어 학습 → [detail-header: ← Unit X 단어]
               → 본문 학습 → [detail-header: ← Unit X 본문]
               → 시험      → [detail-header: ← Unit X 시험]
```

### 7-2. 단어학습_앱.html

| 구분 | 내용 |
|------|------|
| GNB | `<nav id="app-nav">` |
| 탭 | **오늘의 학습(기본)** / 학습 / 테스트 / 단어장 / 종합 (5탭) |
| 상세 | 학습·테스트·단어장·종합 탭 선택 시 GNB 숨김, detail-header 표시 |
| 기본 탭 | 오늘의 학습 (앱 진입 시 자동 활성화) |

**탭별 detail-header 제목**
| 탭 | 기본 제목 | 동적 제목 |
|----|-----------|-----------|
| 학습 | "학습" | `startTodayStudy()` 호출 후 → "Day X 학습" |
| 테스트 | "테스트" | `startTestForDay()` 호출 후 → "Day X 테스트" |
| 단어장 | "단어장" | — |
| 종합 | "종합 테스트" | `startBigTest()` 호출 후 → "종합 테스트 (1~100)" |

**오늘의 학습 탭 콘텐츠**
```
Day X / Y일 (제목, X = primary 색상)
[완료 배지] 오늘 완료! 내일도 파이팅!  ← 학습+테스트 완료 시
[버튼] 오늘 단어 다시 학습 / 오늘 학습 시작
[버튼] 오늘 테스트 보기  ← 학습 완료 후만 표시
[카드] 오늘의 단어 20개 목록 (번호 · 영단어 · 품사 · 뜻 · 발음)
```

**detail-header 우측 요소**
- 학습 탭 → `round-badge` (도입/자동재생/플래시카드 등)
- 테스트 탭 → `test-prog-label` (0/20)

### 7-3. 리스닝_앱.html

| 구분 | 내용 |
|------|------|
| GNB | `<nav id="app-nav">` (2단: top-row + title-section) |
| 탭 없음 | 홈(회차 목록) ↔ 상세(평가화면) 2화면 전환 |
| 상세 진입 시 | GNB(`app-nav`) 전체 숨김 |
| 상세 헤더 | `session-header-bar` (← 뒤로가기 + 회차명 + 초기화) |

**화면 전환 흐름**
```
홈 (회차 카드 목록)
  └─▶ 평가 화면 (GNB 숨김)
        ├─▶ 완료 → 결과 화면
        └─▶ ← 뒤로가기 → 홈 (GNB 복원)
```

**JS 호출**
```js
showXPFloat(elem, amount) {
  const rect = elem.getBoundingClientRect();
  const float = document.createElement('div');
  float.className = 'xp-float';
  float.textContent = `+${amount} XP`;
  float.style.left = (rect.left + rect.width / 2) + 'px';
  float.style.top  = rect.top + 'px';
  document.body.appendChild(float);
  setTimeout(() => float.remove(), 1000);
}
```
